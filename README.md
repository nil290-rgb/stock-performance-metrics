# stock-performance-metrics
 single_asset: Exploratory analysis for a single stock. Computes Sharpe, Sortino, Treynor, Calmar, Omega, Beta, Max Drawdown, CAGR, VaR. Designed for interpretability and understanding individual stock behavior.   
 multi_asset:  Generalized, scalable analysis for multiple equities. Adds market-relative metrics such as beta vs S&amp;P 500, correlations, and batch processing. Designed for portfolio-level insights and batch analysis.
